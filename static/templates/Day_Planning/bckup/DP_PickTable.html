Day Planning Pick Table PLEASE SCAN
Quick Help
Actions to Perform
1. Find your lot in the pick table using Batch ID or Plating Stock No.
2. Click Tray Scan in the Action column — tray scan modal opens.
3. Scan the tray barcode — tray ID fills in automatically.
4. Enter the quantity for each scanned tray.
5. Check Top Tray on the topmost tray in the stack.
6. If not finished, click Save Draft — reopen anytime to continue.
7. Once all trays are scanned, click Submit — lot moves to Completed Table.

Day Planning → Input Screening → Brass QC → IQF → Brass Audit → Jig Loading → In-Process → Jig Unloading → Nickel Inspection → Nickel Audit → Spider Spindle
Stuck? — Save as Draft → Close modal → Reopen → Re-scan → Validate again. If issue persists, note the Batch ID + Tray ID and raise with supervisor.
Raise a complaint — share Batch ID, Lot ID, Tray ID, module, step, and error text.
